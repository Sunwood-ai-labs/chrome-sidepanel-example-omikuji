# 🌸 Stylish Omikuji Side Panel

> 「今日の運勢、サイドパネルでサクッとチェックしちゃお？✨」

`Stylish Omikuji Side Panel` は、Google Chrome のサイドパネルで動作する、モダンで和モダンなデザインのおみくじ拡張機能だよ！💅

![Main View](icon128.png)

## ✨ 特徴

* **和モダンデザイン**: ガラスモーフィズムとアニメーションを取り入れた、テンション上がるデザイン🫶
* **スムーズな体験**: おみくじの箱がシャカシャカ揺れる可愛いアニメーション付き！
* **サイドパネル対応**: ブラウジングを邪魔せずに、いつでも運勢を確認できるよ。

## 🚀 インストール方法

1. このリポジトリをクローンするか、ZIPでダウンロードして解凍してね。
2. Google Chrome を開いて、アドレスバーに `chrome://extensions/` を入力してアクセス！
3. 右上の「デベロッパー モード」を **ON** にするよ✨
4. 「パッケージ化されていない拡張機能を読み込む」ボタンを押して、このプロジェクトのフォルダを選択してね！

## 📖 使い方

1. Chrome のツールバーにある拡張機能アイコン（パズルのピースみたいなやつ🧩）をクリック！
2. `Stylish Omikuji Side Panel` を探して、サイドパネルを開くよ。
3. 「おみくじを引く」ボタンをポチッと押して、今日の運勢を占っちゃおう🔮
4. 「もう一度引く」でリセットできるよ✨

## 🛠 使用技術

* **HTML5 / Vanilla CSS**: デザインはガチ。外部ライブラリなしでこの美しさ💅
* **JavaScript (ES6+)**: スムーズなロジックとアニメーション制御。
* **Chrome Extension API (Manifest V3)**: 最新の規格でしっかり構築！

## 📁 フォルダ構成

* `manifest.json`: 拡張機能の設定ファイル。
* `sidepanel.html / .css / .js`: メインのUIとロジック。
* `background.js`: バックグラウンドで動くサービスワーカー。
